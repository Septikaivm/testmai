Soal & Jawaban Pengetahuan Umum



  1. Apa yang kamu ketahui tentang Version Control, (gunakan bahasa sendiri)

    • Sistem yang mencatat semua perubahan yang pada file atau sekumpulan file , jadi dengan demikian kita dapat memanggil versi spesifiknya dilain waktu.


  2. Jelaskan apa itu GIT

    • Tool yang sering digunakan dalam proyek pengembangan software.
    • Salah satu software yang bisa melakukan vcs.
    • Sebuah software untuk mengelola perubahan file di dalam folder (Repository/repo)
    • Git akan menyimpan setiap history perubahannya menggunakan serangkaian commit.
    • Git Command (local):
      ^ $ git init, untuk menginisialisasi repo git dikomputer. ^ $ git add , untuk menambahkan file.
      ^ $ git status, untuk mengetahui status repo kita apakah ada perubahan atau tidak.
      ^ $ git commit, untuk melakukan commit.
      ^ $ git config, untuk memasukkan konfigurasi kedalam git.
      ^ $ git branch, untuk bikin branch.
      ^ $ git help, untuk mengetahui cara pakai sebuah perintah.


  3. Apa yang kamu ketahui tentang Framwork khususnya web Framework (gunakan bahasa sendiri).

    • sebuah software yang berisi kerangka terstruktur untuk memudahkan para programmer membuat aplikasi atau web
      yang isinya adalah berbagai fungsi, plugin, dan konsep sehingga membentuk suatu sistem tertentu. Dengan menggunakan framework,
      sebuah aplikasi akan tersusun dan terstruktur dengan rapi.
    • contoh framework :
      ^PHP FRAMEWORK (Laravel, Code Igniter (CI)
      ^CSS FRAMEWORK (Bootstrap)
      ^JS FRAMEWORK (Angular-JS)


  4. Jelaskan istilah-istilah berikut: REST API, SQL, HTTP, UI Framework (gunakan bahasa sendiri)

    • Rest API : Antarmuka yang berupa kumpulan fungsi yang dapat dipanggil atau dijalankan oleh program lain.
    • SQL :Sekumpulan perintah khusus yang digunakan untuk mengakses data dalam database relasional.
    • HTTP : Sebuah protokol jaringan lapisan aplikasi yang digunakan untuk sistem informasi.
    • UI Framework : Framework open source yang bisa diintegrasikan dengan mudah kedalam project yang sedang kita buat.


  5. Apa yang dimaksud dengan RDBMS dan noSQL, jelaskan perbedaannya

    • NoSql : Suatu gerakan untuk mempromosikan penyimpanan data non-relasional.
    • (RDBMS) : Relational database management system sebuah program kompute yang didisain untuk mengatur sebuah basisdata sebagai sekumpulan data yang disimpan secara terstruktur.
    • Perbedaannya :
      Karena basisdata SQL bersifat relasional, dari sinilah asal nama sistem manajemen basisdata relasional (RDBMS),
      basisdata NoSQL adalah non-relasional atau terdistribusi. Basisdata SQL berbasis tabel sedangkan basisdata NoSQL
      berbasis dokumen, basisdata grafik, penyimpanan kolom lebar, atau pasangan kunci-nilai. Bahasa query terstruktur
      adalah asal SQL (Structured Query Language) mendapatkan namanya. Dalam NoSQL DB, fokus untuk permintaan
      berada pada koleksi dokumen. Hal ini kadang-kadang disebut Unstructured Query Language (UnQL). Sintaks UnQL bervariasi dari satu basisdata ke basisdata yang lain
